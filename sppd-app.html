SPPD Application

Mobile Application

Aplikasi pemerintahan untuk scheduling dan manajemen Surat Perintah Perjalanan Dinas (SPPD). Dilengkapi dengan:
  • Sistem autentikasi aman.
  • Dashboard interaktif ringkasan SPPD.
  • Form pengajuan user-friendly dengan validasi.
  • Daftar riwayat dengan fitur filter dan pencarian.
  • Pelaporan hasil dinas dan unggah dokumen.
  • Notifikasi push untuk persetujuan.
Tech Stack: Flutter
